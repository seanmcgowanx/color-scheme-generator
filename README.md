# Color Scheme Generator

A simple yet powerful tool to generate and view color schemes right in your browser. Whether you're a designer looking for the perfect palette, a developer in need of some UI inspiration, or just curious about color combinations, this color scheme generator is designed to help. It leverages the power of the Color API to dynamically fetch and display color schemes based on your choice.

## Features

- **Dynamic Color Scheme Generation**: Select a base color and a scheme type (e.g., Monochrome, Analogic, Triad) to generate a color palette.
- **Copy to Clipboard**: Click on any color in the generated scheme to copy its HEX code to your clipboard.
- **Responsive Design**: Built with Bootstrap for a responsive design that adjusts to your device's screen size.
- **Elegant and User-Friendly Interface**: A minimalistic design powered by custom CSS for a smooth user experience.

## How to Use

1. **Select a Base Color**: Use the color picker to choose your base color.
2. **Choose a Scheme Type**: Select a scheme type from the dropdown menu (e.g., Monochrome, Analogic-complement).
3. **Generate Color Scheme**: Click the "Get color scheme" button to fetch and display the color scheme.
4. **Copy HEX Values**: Click on any of the displayed colors to copy its HEX value to your clipboard.

## Setup

1. **Clone the Repository**

git clone https://color-scheme-generator

2. **Navigate to Project Directory**

cd your-project-directory

Replace `your-project-directory` with the name of the directory where you've saved or cloned the Color Scheme Generator project.


3. **Open `index.html` in Your Browser**

Simply open the `index.html` file in your favorite browser to start using the color scheme generator.
